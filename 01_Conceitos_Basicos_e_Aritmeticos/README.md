# 📚 01. Conceitos Básicos e Aritméticos

Este módulo contém os meus primeiros passos na linguagem Java, focando na sintaxe, operações matemáticas e manipulação simples de dados.

### 📋 O que você vai encontrar aqui:

* **Estrutura Básica:**
    * `OlaMundo`: O clássico ponto de partida.
    * `QualSeuNome`: Entrada e saída simples de dados (Scanner).
* **Operações Aritméticas e Lógica Simples:**
    * `Soma`, `Media`: Cálculos matemáticos básicos.
    * `NumeroAntecessorSucessor`, `TercaParteDeUmNumero`: Manipulação de variáveis numéricas.
* **Mini-Problemas Práticos:**
    * `EmprestimoParcelarJuros`: Cálculo de juros simples.
    * `NomeSalarioDoFuncionario`: Modelagem simples de dados de funcionários.
    * `ParedeTinta`: Cálculo de área e quantidade de tinta.

### 🛠 Tecnologias e Conceitos:

* Java (Scanner, tipos primitivos como `int`, `double`, `float`).
* Leitura e escrita no console.
* Operadores aritméticos (+, -, \*, /).

# 🧮 02. Matemática e Math

Este módulo marca a transição da aritmética simples para o uso de funções matemáticas mais avançadas. O objetivo foi explorar a classe nativa `java.lang.Math`, que oferece ferramentas para trigonometria, raízes, potências e outras operações complexas essenciais para o desenvolvimento backend.

### 📋 Projetos Neste Módulo:

* **`AnguloSeno`**
    * **Descrição:** Lê um ângulo em graus do usuário e calcula o seu Seno.
    * **Destaque Técnico:** Demonstração prática do uso da classe `Math`. Como o Java realiza cálculos trigonométricos em Radianos, este projeto mostra a etapa crucial de conversão de Graus para Radianos usando `Math.toRadians(anguloGraus)`, seguida pelo cálculo do seno com `Math.sin(radiano)`.

### 🛠 Tecnologias e Conceitos Explorados:

* **Lógica e Tipagem:** Uso do tipo `double` para garantir a precisão necessária em cálculos trigonométricos.
* **Classe `Math`:** Utilização de métodos estáticos como `toRadians()` e `sin()`.
* **Formatação:** Uso da classe `DecimalFormat` com o padrão `#.#` para exibir o resultado final de forma limpa, focando apenas no valor significativo.

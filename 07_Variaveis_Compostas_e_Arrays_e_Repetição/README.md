# 📦 07. Variáveis Compostas (Arrays) e Repetição

Este módulo marca a transição para o uso de **Estruturas de Dados**, focando no gerenciamento de coleções de informações de forma organizada. O objetivo principal foi aprender a manipular múltiplos valores sob um único identificador e aplicar laços de repetição para processar esses dados em massa.

## 📝 Projetos Neste Módulo:

### 🗳️ Sistema de Votação (Evolução de Lógica)
* **`BbbParedaoProgramaLimpo`** (⭐ Destaque de Engenharia):
    * **Descrição:** Simulador avançado de votação para o BBB 26 com pesos diferenciados (CPF e Torcida).
    * **Destaque Técnico:** Aplicação de **Constantes** (`static final`), modularização com **Métodos** e gerenciamento de dados em paralelo usando Arrays.
* **`BbbParedao`**:
    * **Descrição:** Primeira versão do sistema de votação focada em cálculos aritméticos e médias ponderadas.
    * **Destaque Técnico:** Uso de `Long.parseLong` e tratamento de Strings com `.replace()` para lidar com grandes volumes de votos inseridos pelo usuário.

### 🎓 Gestão Acadêmica e Avaliação
* **`GabaritoNotaseMedia`**:
    * **Descrição:** Sistema automático de correção de provas e cálculo de média da turma.
    * **Destaque Técnico:** Uso de **Arrays de diferentes tipos** (`char[]` para gabarito e `double[]` para notas) e implementação de laços aninhados para comparação de dados.

## 🛠️ Tecnologias e Conceitos Explorados:

* **Arrays Unidimensionais (Vetores):** Armazenamento eficiente de listas de nomes, votos e notas.
* **Modularização:** Separação de responsabilidades em métodos estáticos (`somar`, `calcularPercentual`).
* **Clean Code:** Uso de constantes para facilitar a manutenção e legibilidade do código.
* **Formatação de Saída:** Uso do `printf` para criar tabelas de resultados alinhadas no console.

# 🛠️ 09. Funções e Modularização

Este módulo explora a criação de métodos estáticos (funções e procedimentos) para organizar a lógica do programa. O foco foi aprender a passar parâmetros, retornar valores e entender o escopo de variáveis, permitindo a criação de códigos mais limpos e reutilizáveis.

## 📝 Projetos Neste Módulo:

### 🧮 Lógica Matemática com Funções

* **`SomaFuncao`**:
    * **Descrição:** Um exemplo básico para entender a estrutura de uma função que recebe dois parâmetros e retorna a soma.
* **`FatorialFuncao`** (⭐ Destaque de Recursividade Lógica):
    * **Descrição:** Calcula o fatorial de um número utilizando uma função dedicada.
    * **Destaque Técnico:** Uso do tipo `long` para suportar grandes resultados e separação clara entre a interface de entrada e o processamento matemático.
* **`VerificarParImparFuncao`**:
    * **Descrição:** Uma função que recebe um inteiro e retorna uma String indicando se o valor é "PAR" ou "ÍMPAR".

### 🌀 Sequências e Algoritmos

* **`FibonaciProcedimento`** & **`SequenciaDeFibonacci`** (⭐ Destaque de Algoritmos):
    * **Descrição:** Implementação da famosa sequência de Fibonacci utilizando funções para calcular o próximo termo.
    * **Destaque Técnico:** Demonstração de passagem de parâmetros para manter o estado da sequência e controle de laços de repetição dentro de métodos.

### 🔠 Manipulação de Strings

* **`AnalisadorDeNomes`** (⭐ Destaque de Processamento de Dados):
    * **Descrição:** Um sistema completo que analisa o nome do usuário, conta letras, inverte o texto e busca códigos ASCII.
    * **Destaque Técnico:** Uso intensivo de métodos da classe `String` como `charAt()`, `substring()`, `toUpperCase()` e manipulação de índices com laços `for`.

## 🛠️ Tecnologias e Conceitos Explorados:

* **Métodos Estáticos:** Criação de funções que podem ser chamadas diretamente pelo `main`.
* **Retorno de Valores (`return`):** Implementação de lógica que processa dados e devolve resultados para o fluxo principal.
* **Passagem de Parâmetros:** Comunicação entre diferentes blocos de código através de argumentos.
* **Tabela ASCII:** Conversão de caracteres em valores numéricos para análise de baixo nível.

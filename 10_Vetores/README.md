# 📦 10. Vetores (Arrays) e Estruturas de Dados

Este módulo explora o armazenamento de múltiplos valores em uma única variável. O foco foi entender a indexação, manipulação de coleções, algoritmos de busca e a introdução ao uso de matrizes (vetores multidimensionais).

## 📝 Projetos Neste Módulo:

### 📑 Gestão de Dados e Listagens

* **`TesteVetor`**: 
    * **Descrição:** Exercício fundamental para entender a declaração e o preenchimento de arrays unidimensionais.
* **`Turma`** (⭐ Destaque de Processamento de Médias): 
    * **Descrição:** Lê nomes e notas de alunos, calcula a média da turma e identifica quem está acima dela.
    * **Destaque Técnico:** Uso de múltiplos vetores paralelos para associar nomes a valores numéricos.
* **`ValoresPares`**: 
    * **Descrição:** Identifica e conta quantos números pares foram inseridos em um vetor de 7 posições.

### ⚽ Lógica de Jogos e Campeonatos

* **`CinemaAssento`** (⭐ Destaque de Sistema de Reserva): 
    * **Descrição:** Simula um sistema de reserva de cadeiras de cinema com representação visual no console.
    * **Destaque Técnico:** Uso de vetor do tipo `boolean` para controle de estados (ocupado/vazio) e lógica de validação de entradas repetidas.
* **`ConfrontosDaRodada`** (⭐ Destaque de Algoritmo Combinatório): 
    * **Descrição:** Gera automaticamente uma tabela de jogos de ida e volta entre times cadastrados.
    * **Destaque Técnico:** Implementação de laços aninhados (`for` dentro de `for`) para realizar combinações entre os elementos do vetor.

### 🧪 Algoritmos e Filtros

* **`SoComC`**: 
    * **Descrição:** Filtra uma lista de nomes e exibe apenas aqueles que começam com a letra "C".
* **`VetorOrdem`** & **`VetorOrdemSemArraysSort`** (⭐ Destaque de Algoritmos de Ordenação): 
    * **Descrição:** Compara a facilidade de usar o método `Arrays.sort()` com a implementação manual de um algoritmo de ordenação (Bubble Sort simplificado).
    * **Destaque Técnico:** Domínio da lógica de "troca de valores" (variável auxiliar) para organizar elementos manualmente.

### 🎓 Sistemas Escolares Avançados

* **`CorrigindoAsProvasComGabarito`** (⭐ Destaque de Matrizes): 
    * **Descrição:** Sistema completo que cadastra um gabarito, lê respostas de vários alunos e gera notas finais.
    * **Destaque Técnico:** Introdução às **Matrizes** (`double[][]`) para armazenar dados multidimensionais (Alunos x Questões).

## 🛠️ Tecnologias e Conceitos Explorados:

* **Indexação:** Compreensão de que arrays em Java iniciam na posição `0`.
* **Laço For-Each:** Uso da sintaxe simplificada `for (int valor : vetor)` para percorrer coleções.
* **Algoritmos de Ordenação:** Entendimento de como o computador organiza dados internamente.
* **Manipulação de Matrizes:** Organização de dados em linhas e colunas para problemas complexos.
